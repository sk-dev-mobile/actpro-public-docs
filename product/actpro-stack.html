Стек и структура проекта

Общее решение

Проект строится как монорепозиторий с двумя основными приложениями:

  • web-панель для администрирования и рабочих сценариев;
  • отдельный серверный API.

Web-панель

Основной стек

  • React
  • Vite
  • TypeScript
  • React Router

Стили и компоненты

  • Tailwind CSS
  • shadcn/ui

Сервер

API и приложение

  • FastAPI
  • Python

Работа с базой

  • PostgreSQL
  • SQLAlchemy 2.0 async
  • Alembic
  • asyncpg

Служебные сервисы

  • Redis для ограничения попыток входа, временных блокировок и служебных данных;

Доступ

  • короткий access JWT;
  • долгая refresh session на сервере;
  • refresh session можно отзывать.

Структура проекта

apps/ backend/ app/ main.py core/ common/ modules/ auth/ organizations/ users/ experts/ reports/ migrations/ tests/ frontend/ src/ app/ api/ lib/ pages/ components/ ui/ features/ public/

Подход к серверу

Сервер строится как модульный монолит.

Один backend разделяется по бизнес-модулям, без микросервисов и без лишней архитектурной сложности.

Зафиксировано

React Vite TypeScript React Router Tailwind CSS shadcn/ui FastAPI PostgreSQL SQLAlchemy async Alembic asyncpg Redis modular monolith
