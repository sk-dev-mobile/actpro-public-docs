PDF генератор: DTO и HTML

Две точки, если мобильный клиент хочет совпасть с тем, что уходит в печать на сервере.

1. Актуальный DTO для генератора

В монорепозитории продукта (ветка main):

  • JSON Schema: docs/mobile/act-render-dto.schema.json
  • Текст по полям: docs/mobile/act-render-dto.md
  • Pydantic-модели: apps/backend/app/modules/acts/render/schemas.py
  • Сборка из сохранённого акта: apps/backend/app/modules/acts/render/mapper.py

Схема в репо, не на этом портале. Если доступа к git нет, попросите у команды выгрузку этих файлов.

2. HTML шаблон генератора

Папка в том же репозитории: apps/backend/app/modules/acts/pdf/templates/

  • Приёмка: act.html, стили act.css
  • Экспертиза: act_expertise.html, стили act_expertise.css

Jinja-рендер: apps/backend/app/modules/acts/pdf/renderer.py
