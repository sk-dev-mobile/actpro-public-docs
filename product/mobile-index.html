Mobile API Scenarios

Короткие сценарии для mobile developer: вход эксперта, получение структуры акта, загрузка фото, отправка и просмотр актов.

Prod base URL: https://api.lgtime.ru

Auth: Authorization: Bearer <access_token>

Все endpoint-ы ниже даны относительно /api/v1.

ready
PIN-вход эксперта, refresh token и проверка сессии через /users/me.

Каталог и структура акта

ready
Получение каталога, разбор standard_room_id, group_ids и local_groups.

Фото и отправка акта

ready
Upload временных фото, идемпотентность и финальный POST /acts.
